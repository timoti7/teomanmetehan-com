# teomanmetehan.com

Personal portfolio/CV website for Teoman Metehan.

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JS (minimal animations)

## Structure
```
teomanmetehan.com/
├── index.html      # Single-page portfolio
├── assets/         # Images (placeholder for photo)
└── README.md
```

## Sections
1. **Hero** - Name, title, intro
2. **About** - Background, Sehrikeyif story, current focus
3. **Experience** - TVI/Sehrikeyif, Teknik Yapi, International
4. **Research** - PhD research, SAGE Open publication
5. **Education** - PhD, MBA, MSc, BSc
6. **Contact** - Email, LinkedIn, Location

## Local Development
Simply open `index.html` in a browser. No build step required.

## Deployment
Designed for static hosting (Railway, Netlify, Vercel, etc.)

## Notes
- Dark theme with blue accents
- Mobile responsive
- LinkedIn is primary CTA
- No blog/dynamic content - "For more, visit LinkedIn"
